Charter carries every word a person actually reads or writes — one considered, warm voice throughout. The interface's own chrome (labels, timestamps) uses the device's native system font on purpose, so it recedes rather than competes for attention.

Space between turns mirrors conversational rhythm, not a uniform grid — see visual-design-system-v1.md's spacing philosophy.

Motion, elevation, radius

One duration (260ms, ease-out) used everywhere motion appears. Elevation is almost never used — the product reads as flat pages, not floating cards; the one exception is the Understanding panel, given the barest lift (a 1px hairline shadow) to mark it as something you can hold and inspect. Corner radius is a single, nearly imperceptible 3px, applied only to that panel and the composer's focus ring — never a stylistic "rounded-everything" signature.
